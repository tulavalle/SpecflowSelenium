# JavaScriptCore

<details>
<summary><b> ExecuteCommand </b></summary>

Executa Comando JS.

Exemplo de uso:

```c#
  JavaScriptCore.ExecuteCommand(driver, comandoDesejado);
```

<details>
<summary><b> ExecuteCommand </b></summary>

Exemplo de uso:

```c#

```

</details>


</details>
