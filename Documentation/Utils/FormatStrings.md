# FileActions

<details>
<summary><b> GetPathFileLocal </b></summary>

Obtém o caminho local de um arquivo.

Exemplo de uso:

```c#
  FileActions.GetPathFileLocal("../PastaArquivos");
```

</details>

<details>
<summary><b> ReadFile </b></summary>

Realiza a leitura de um arquivo.

Exemplo de uso:

```c#
  FileActions.ReadFile("../PastaArquivos");
```
</details>
