# Enumerator

<details>
<summary><b> GetDescription </b></summary>

Obtém a descrição do atribudo do enumerador.

Exemplo de uso:

```c#
  var expectedColor = Enumerator.GetDescription(expectedEnumColor);
```

</details>

```c#

```

</details>
